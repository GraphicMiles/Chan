Step 6 — Resolving
Resolving Media...
The server is solving the O2TV captcha and extracting the CDN URL.
Solving captcha via Groq AI
This takes 5-8 seconds
1. GET episode page → find /download/{id} links
2. GET /areyouhuman.php → get captcha image
3. Send captcha to Groq vision API
4. POST captcha text → get CDN redirect
5. Success → create room → navigate
6. Fail → show error, no room created
Two outcomes
Success → video player plays. Failure → error, no room.
